How React works:
React are mostly used for single page application.
Ekta component ba page er HTML parse hoye DOM e DOMTree and CSS parse hoye CSSOM style rules create kore. CSSOM and DOMTree attach hoye RenderTree craete kore. RenderTree Layout phase er moddhye diye zay. Layout phase e component ba ekta page e kun JSX element kuthay display hobe ta nirdharito hoy. JSX element placed howar por CSSOM diye DOM Paint hoy. Obosheshe amra browser e DOM ba application dekhte pai ebong bivinno interaction kori.
Kintu application er ekta interaction e zodi kuno state change hoy tahole sein state er nicher component ebong sei state use kora sob component gulu notun kore re-Render hoy. Ekhane bivinno JSX element initialization, update and mount/unmout ghote. Tobe re-Render process DOMTree and CSSOM theke notun kore shuru hoy ebong browser e dekhanu te shesh hoy.
React ei re-Render process sohoj korar jonno VirtualDOM use kore. Application e kuno interaction e state change howar ager JSX structure and upcoming JSX structure er moddhye difference kore React re-Render process complete kore shudhu matro proujonio JSX Display kore. React VirtualDOM theke state change er ager ebong porer obosta Diffing/Reconciliation algorithm diye findout kore and actual DOM e surgically change kore dey. Tarpor shudhumatro change howya JSX re-Render->Paint->Display hoy.
