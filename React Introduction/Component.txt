Keeping Components Pure: Single Responsibility Priciple
Pure function only perform a caluculation and nothing more, and return it's own JSX.
Local mutation kora zabe, kintu global mutation kora zabe na.
Pure Component/function same props er jonno same JSX return korbe.
Pure component shudhumatro nijer kaj kore.
Same input er jonno same output pawya zay.
